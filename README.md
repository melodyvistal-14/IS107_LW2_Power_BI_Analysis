Guide Questions
1. Why is removing duplicates important in data analysis?
   - Removing duplicates is important because it prevents the same data from being counted more than once and keeps the results accurate.
2. What problems can missing values cause in reports?
   - Missing values can cause incorrect calculations and make reports incomplete or misleading.
3. How does correct data typing affect calculations?
   - Correct data typing ensures numbers, dates, and text are processed properly so calculations work correctly.
4. What naming conventions improve data readability?
   - Using clear names, lowercase letters, and underscores (like order_id) makes data easier to read and understand.
